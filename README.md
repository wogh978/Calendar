# Calendar
### 달력과 일정을 같이 보여주는 앱을 만들고자 하였습니다.


### 남은 과제
* 오늘 날짜 나타내기
* 공휴일 데이터 불러오기
* 일정 추가할 때 datepicker 을 이용하여 달력에 표시
* 다음 달로 넘어갈 때 일정을 초기화
* 달력에서 특정 날짜를 클릭하면 일정 추가 가능
* 일정에 대한 미리 알람기능
